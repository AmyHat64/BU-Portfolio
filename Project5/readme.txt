Overview
This project demonstrates querying relational databases using SQLite. The petsDB dataset was used to explore relationships and retrieve insights through SQL commands.
Dataset
•	petsDB relational database
•	Tables include persons, pets, and relationships
Methods
•	Establishing database connections
•	Running SQL queries to count, group, and filter records
•	Extracting and summarizing insights
Results
Queries identified age group distributions, relationships between people and pets, and other demographic insights.
Tools
SQLite, Python (sqlite3 library)
Future Work
•	Build visualizations of query results
•	Create interactive dashboards from SQL outputs
•	Explore more advanced SQL joins and subqueries

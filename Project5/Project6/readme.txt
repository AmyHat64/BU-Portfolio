Overview
This project analyzes the NFL careers of Heisman Trophy winners using historical player statistics. It compares expectations of top college athletes with their professional performance.
Dataset
•	NFL Player Stats (1922–2022) CSV file
•	Focused on Heisman winners post-1999
Methods
•	Data cleaning and wrangling in pandas
•	Feature renaming and restructuring
•	Filtering Heisman winners and analyzing their performance metrics
•	Visualizing stats with Matplotlib
Results
The project highlights the variance in professional success among Heisman winners, showing that not all award recipients sustain high performance in the NFL.
Tools
Python, pandas, Matplotlib, SQLite
Future Work
•	Add predictive modeling for career longevity
•	Compare Heisman winners against non-Heisman peers
•	Expand to positional analysis (QB, RB, WR, etc.)
